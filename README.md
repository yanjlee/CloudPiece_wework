# CloudPiece_wework

根据 JoysKang 的 https://github.com/JoysKang/CloudPiece ，修改而来。

使用企业微信的自建应用接收消息。

目前适配了文本消息，其他类型的消息因需要服务器转存后才能推送到notion，暂时不支持。
